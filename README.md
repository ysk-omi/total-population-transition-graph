#

## API

[RESAS-API](https://opendata.resas-portal.go.jp/)

## 開発環境

- Next.js
- TypeScript
- ESLint
- Prettier
- Sass
- [Highcharts](https://www.highcharts.com/)
- Jest

## ローカル開発手順

RESAS API から必要な情報を取得し、,env ファイル内の項目を埋めて、`.env`にリネームして使用してください。

### パッケージインストール

```
npm ci
```

### 起動

```
npm run dev
```

### ビルド

```
npm run build
```

### エクスポート

```
npm run export
```
